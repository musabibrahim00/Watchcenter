RIGHT CONFIGURATION PANEL

The right panel must reuse the existing panel / card styling.

Default state:

Title:
Select Module Type

Below title add a search bar using the shared input component.

Below search show a list of module types.

Each module type should be a clickable card row using the same card style.

Modules include:

Action
Execute a prebuilt or custom script inline.

Trigger
Wait for an event or condition before continuing.

Checkpoint
Pause workflow for human review or input.

Flow
Reuse an existing workflow as a subflow.

For Loop
Repeat actions for each item in a list.

While Loop
Run actions repeatedly while condition remains true.

Branch to One
Continue with a single path matching set conditions.

Branch to All
Continue through multiple paths in parallel.

Each row includes:
Icon
Title
Short description
Chevron on right
